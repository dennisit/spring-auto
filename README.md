# spring-auto

安装
---------------------------

`spring-auto-test` module仅是为了测试而存在不必安装到本地maven仓库

```
git clone git@github.com:yingzhuo/spring-auto.git
cd spring-auto/
mvn clean install -pl '!spring-auto-test' -Dmaven.test.skip=true
```
